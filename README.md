# CAD Camera Controls

Standalone CAD-style camera controls for React Three Fiber.

## Exports

- `CADCameraControls`
- `CADCameraControlsProps`
- `CADCameraControlModifierKey`

## Local development

```bash
npm install
npm run typecheck
```

## Three.js-style HTML example

From this repo root, serve static files and open the example page:

```bash
npx serve .
```

Then open:

- `http://localhost:3000/examples/misc_controls_cad_camera_controls.html`

The page uses three.js examples formatting (`examples/main.css`) and includes:

- `lil-gui` tuning panel
- origin cube at `(0,0,0)`
- origin axes helper
- grid helper
